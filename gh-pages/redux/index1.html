第二章：上传电子笔记

第一步，先安装

npm install gitbook-cli -g

然后自己创建一个自己的笔记文件,名字可以自己起

mkdir my-note

切换到创建的文件夹下

cd my-node

执行命令

gitbook init

最后运行

gitbook serve

在地址栏上写地址直接访问
